## Installation
```
./setup.sh
```

## Preprocessing

Assuming the PokeAPI sprites file is in the same folder as preprocessing.py
```
python preprocessing.py sprites-master/sprites <path to target directory>
```

## Running Examples
```
source venv/bin/activate
python3 <script>.py
```
